# 🤖 AI Customer Analytics Platform

## 📌 Overview

AI Customer Analytics Platform is an intelligent business analytics system developed using Python, Streamlit, and Machine Learning. It helps businesses understand customer behavior through segmentation, prediction models, and AI-driven insights.

---

## 🚀 Features

### 📊 Dashboard

* Upload Customer Dataset
* KPI Cards
* Dataset Preview
* Statistical Summary
* Data Cleaning Report

### 📈 Business Visualizations

* Histograms
* Pie Charts
* Bar Charts
* Scatter Plots
* Heatmaps
* Box Plots

### 🤖 Customer Segmentation

* K-Means Clustering
* Cluster Visualization
* Purchase Prediction
* Recommendation Engine

### 💡 AI Insights

* AI Business Score
* Customer Churn Prediction
* Customer Lifetime Value Prediction
* Future Purchase Prediction
* AI Marketing Strategy
* Executive Dashboard

### 📄 Reports

* CSV Export
* PDF Export
